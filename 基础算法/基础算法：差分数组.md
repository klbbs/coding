# 基础算法：差分数组

- ### 作用点：在数组一定范围内[l,r]同时加上同一个数，这样需要操作r-l+1次，但是使用差分数组，将只需要操作两次。

- ### 原理：差分数组S是用于记录原数组数据增量的数组，基本定义数组中第i个元素值为==S~i~=A~i~-A~i-1~==（A为原数组)

  ## 示例：在数组一定范围内均增加k,范围[l,r]

	``````c++
	int S[N]{0};
	for(int i=1;i<N;i++){
		S[A[l]] += k;
	    S[A[r+1]] -= k;
	}
	``````
#### 	这样就记录这次操作，如果想加入原数组，就是对应区间遍历+=

``````cpp
for(int i=1;i<N;i++){
	S[i] += S[i-1];
}
``````
